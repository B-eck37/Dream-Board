## Project created for DevMountain Personal Project



Routes:

'/' - Login Component - Initial Page

'/home' - Home Component - User redirect after Login - Displays general 'Suggested Images'. Provides ability to save.

'/myaccount' Account/Admin Component - Displays User Account Info (Basic) - Uses Join statement for DB to show User image URL total.

'/mydreams/:userId' MyDreams Component - Displays user created boards (Personalized)

'/searchdreams' Search Component - Provides user opportunity to search new images. Provides Save ability.
